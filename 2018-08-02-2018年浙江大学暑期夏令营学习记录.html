2018年浙江大学暑期夏令营学习记录

Posted on 2018-08-02 | Edited on 2018-08-07 | In 随笔 | Comments: | Views:
Symbols count in article: 1.5k | Reading time ≈ 3 mins.

2018 CVPR Best Paper Taskonomy

  • 完成论文全文阅读，理解论文所做工作，Taskonomy构建思路、步骤
  • 初步了解迁移学习思想
  • 理解Beam Search算法思想以及算法流程
  • 了解Boolean Integer Programming：0-1整数规划问题(通过0-1背包问题理解)
  • 下载项目源码，以及模型和模型参数，在自己的测试图片运行代码
  • 尝试阅读源码，效果不佳

迁移学习

  • 了解迁移学习的目的与作用，理解迁移学习的定义
  • 着重了解了深度迁移学习的思路以及方法，理解深度迁移学习的损失表示，以及几个核心方法的架构、损失函数。
  • 大致了解深度对抗迁移网络
  • 基于花朵数据集完成最简单的深度迁移学习:finetune，从深度卷积网络VGG迁移，完成识别五种花的任务。
  • 尝试阅读深度网络自适应方法源码，效果不佳
  • 尝试理解MMD方法…
  • 阅读论文DCC，无法理解RKHS(reproducing kernel Hilbert space)
  • 阅读论文DAN，数学公式无法理解，如何在O(n)时间复杂度求解MK-MMD

基础神经网络学习

对抗生成网络(GAN)

  • 理解对抗生成网络原理，理解损失函数的构成
  • 基于Tensorflow完成最基本形式的生成器、判别器的搭建，以及损失函数的编写
  • 在minist数据集上训练GAN，完成生成手写数字图片的工作
  • 完成LSGAN(Least Squares GAN)网络、以及Deep Convolutional GANs(DCGAN)的搭建以及在minist数据集上的训练

递归神经网络(RNN)

  • 理解基础递归神经网络的原理、损失函数的表示、训练方式
  • 基于Tensorflow完成基础Vanilla RNN的搭建，以及在Microsoft COCO数据集上完成训练，能够完成为图片取标题的任务
  • 尚未完成LSTM

SqueezeNet

  • 阅读论文：Squeezenet: Alexnet-level accuracy with 50x fewer parameters and< 0.5 mb model size
  • 理解网络架构，使用该网络作为特征提取器实现风格迁移

图像风格迁移(Style Transfer)

  • 理解原理，损失函数(内容损失、风格损失、协方差)
  • 使用SqueezeNet作为特征提取器，完成图像风格迁移任务

算法练习

  • 刷LeetCode array专题题目，难度集中在medium,少量hard
  • 二分法专题开始

博客搭建

  • 网站搭建完毕
  • 域名注册，并实名认证完毕
  • 域名解析完成，并提交Google, Baidu
  • 进行SEO优化完毕

Taskonomy 实验结果复现

2018-08-06

  • 开始阅读源码
  • 只提供了task-specified network模型参数！！！
  • 源码只是提供了使用task-specified模型的api！！！
  • 并没有给出根据taskonomy训练的迁移模型！！！

2018-08-07

  • 论文实验思路

    • task-specified network: 基于ResNet-50魔改encoder，针对不同任务进行finetune，使用论文提供的数据集训练，作为迁移学习的源任务，或者baseline
    • 构建transfer function，用于迁移学习，使用task-specified network的representation进行迁移
    • 使用BIP的api计算出迁移策略，根据策略进行迁移
    • 在MIT places和ImageNet上迁移，对比不同数据集上人物之间的关系结构是否还存在
